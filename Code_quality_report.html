print_box() löscht ANSI-Farben aus Inhalt still
game.py : 108–118 · Funktion print_box()
▶

Problem: textwrap.wrap() wird auf den ANSI-bereinigten String angewendet. Das Ergebnis — der bereinigte Text wline — wird dann gedruckt, nicht der Original. Inline-Farben wie {FETT}{CYAN}SDD-Projektdokumentation{RESET} (Zeile 865) gehen damit still verloren. Es gibt keine Fehlermeldung — der Text erscheint nur unformatiert.

Ursache
# Der wrap operiert korrekt auf dem plain-Text —
# aber wline IST der plain-Text, nicht der Original
wrapped = textwrap.wrap(_ANSI_RE.sub("", subline), width=w - 4) or [""]
for wline in wrapped:
    padding = " " * max(0, w - 4 - _display_len(wline))
    print(f"  {color}│{RESET}  {wline}{padding}{color}│{RESET}")
#                        ^^^^^ plain-Text, Farbe weg
Fix
# plain nur für Breiten-Berechnung; Ausgabe mit Original
plain = _ANSI_RE.sub("", subline)
wrapped = textwrap.wrap(plain, width=w - 4) or [""]
for wline in wrapped:
    padding = " " * max(0, w - 4 - _display_len(wline))
    print(f"  {color}│{RESET}  {wline}{padding}{color}│{RESET}")
# Hinweis: Bei Inline-ANSI + Wrapping stimmt die Breite
# nicht exakt — Aufrufer sollten Inline-ANSI sparsam nutzen.

▶

Problem: pause() ist eine UI-Hilfsfunktion, beendet aber das Programm mit sys.exit(0). Das koppelt Exit-Logik in die falsche Schicht, macht Unit-Tests unmöglich und überrascht jeden, der pause() in anderem Kontext aufruft.

— vorher

    if antwort == "x":
        sys.exit(0)
+ nachher

    antwort = input(msg).strip().lower()
    return antwort == "x"  # True = Ende

# An allen ~8 Aufrufstellen:
if pause():
    sys.exit(0)

150-Spalten-Pflicht sperrt Nutzer mit normalen Terminals aus
game.py : 203–212 · Funktion show_intro()
▶

Problem: while terminal_width() < 150: erzeugt einen Endlos-Loop. Auf vielen Laptops ist ein maximiertes Terminal nur 120–140 Spalten breit. Der einzige Ausweg ist "x" zum Beenden — kein Fortfahren trotz kleinerer Ansicht möglich.

Fix: Warnung anzeigen, aber nach Bestätigung trotzdem durchlassen. Der Loop wird zu einer einmaligen Hinweis-Ausgabe ohne Zwang.

Fix-Skizze
if terminal_width() < 150:
    print_centered(f"{GELB}Optimale Darstellung ab 150 Spalten "
                   f"(aktuell: {terminal_width()}){RESET}")
    print_centered(f"{GRAU}Du kannst trotzdem fortfahren.{RESET}")
    antwort = input(f"  [ Enter ]  [x für Ende]  ").strip().lower()
    if antwort == "x":
        sys.exit(0)
    clear_screen()
# kein while — einmalige Warnung, kein erzwungener Loop

    print(f"  {GRAU}(Ctrl+C zum Überspringen){RESET}")
    try:
        time.sleep(pause_before)
        for idx, line in enumerate(lines):
            ...  # bestehende Logik
    except KeyboardInterrupt:
        print()  # Zeile abschließen
        for idx, line in enumerate(lines):
            color = (colors[idx] if colors and idx < len(colors) else "")
            print(color + line + RESET)
