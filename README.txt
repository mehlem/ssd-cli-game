SDD CLI Game
============

Spec-Driven Development interaktiv erleben.
Durchlaufe alle 7 SDD-Phasen und rette das PromptAndPray-Projekt.

Voraussetzung
-------------
Python 3.8 oder neuer muss installiert sein.
Download: https://www.python.org/downloads/

Starten
-------
Windows (Doppelklick):
  start_game.bat       -- oeffnet ein neues PowerShell-Fenster (farbige Emojis)

Windows (Terminal):
  py game.py

Mac / Linux (Terminal):
  python3 game.py

Hinweise
--------
- Fuer optimale Darstellung bitte das Terminalfenster maximieren.
- Das Spiel speichert keinen Fortschritt -- jeder Start ist eine neue Runde.
- Die Fragen werden bei jedem Start zufaellig ausgewaehlt und gemischt.
